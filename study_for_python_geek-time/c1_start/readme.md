#### 运行第一个 python 程序

- python 是解释型语言，需要先转换成字节码，再经过 python 虚拟机执行
- 执行(非交互) 2.py: ``python3 2.python``
- 交互执行(执行后不退出python解释器环境)，要加参数 -i: ``python3 -i 2.py``
