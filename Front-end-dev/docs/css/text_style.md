## 文本样式

区别于字体样式，文本样式作用的是“整个段落”的排版效果.

- 首行缩进(text-indent)
- 水平对其(text-align)
- 文本修饰(text-decoration)
  - none
  - underline
  - line-through
  - overline: 实际上几乎用不到
- 大小写(text-transform)
  - none
  - uppercase
  - lowercase
  - capitalize
- 行高(line-height)
- 间距(letter-spacing & word-spacing)
